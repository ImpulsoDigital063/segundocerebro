# Configurações recomendadas — Obsidian

---

## Plugins essenciais (instalar primeiro)

1. **Daily Notes** (nativo) — cria nota do dia automaticamente
   - Template: `1-ENTRADA/daily/TEMPLATE-DAILY.md`
   - Pasta destino: `1-ENTRADA/daily/`
   - Formato da data: `YYYY-MM-DD`

2. **Dataview** (comunidade) — visualizar dados das notas como tabela
3. **Templater** (comunidade) — templates mais poderosos que o nativo

---

## Estrutura de pastas no Obsidian

Vai aparecer exatamente assim:
```
1-ENTRADA
2-PROCESSAMENTO
3-RETENCAO
4-EXPORTACAO
_config
```

---

## Fluxo de uso diário

1. Abre Obsidian
2. Cria Daily Note (Ctrl+P → "Daily Note")
3. Anota tudo que acontece no dia
4. Quando inbox tiver 5+ itens → processa
5. Atualiza STATUS dos projetos (Impulso, MPN)
6. Quando aprender padrão novo → adiciona em `3-RETENCAO/padroes/`
7. A cada semana → atualiza MEGA-CLAUDE.md
