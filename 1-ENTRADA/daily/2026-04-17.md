## O que foi feito — Deploy Impulsodesign

### Migração técnica completa
O app rodava só local porque:
1. **Storage em `.data/` (filesystem)** — Vercel é read-only, não dá pra salvar carrossel em JSON local
2. **Claude Code CLI (`USE_CLAUDE_CODE=1`)** — CLI não existe em servidor de produção
3. **`middleware.ts`** — Next 16 renomeou pra `proxy.ts` (função `proxy`)
4. **Leitura do segundo cérebro via `C:/Users/DELL/segundo-cerebro`** — esse path não existe na Vercel

Todos os 4 foram resolvidos:
- `lib/kv.ts` — client Upstash Redis via `@upstash/redis`
- Todas as storages (`carrosseis`, `pautas`, `brand-brain`, `templates`, `cost-tracker`) reescritas com prefixos (`carrossel:`, `pauta:`, `brain:`, etc.)
- `scripts/migrate-to-kv.ts` — subiu os 2 carrosseis locais + cost log pro Redis
- `lib/storage/segundo-cerebro.ts` — agora lê via GitHub Contents API (`raw.githubusercontent.com`) com cache em memória 5min
- `middleware.ts` → `proxy.ts` (função renomeada pra `proxy`)
- `vercel.json` — região `gru1` (São Paulo) + `maxDuration: 60` nas rotas IA pesadas

### Infra Vercel
- Projeto `impulsodesign` conectado ao GitHub `ImpulsoDigital063/impulsodesign`
- Upstash Redis (plan Free: 500k comandos/mês) em região Brazil East
- 11 env vars configuradas: `IMPULSO_PASSWORD`, `ANTHROPIC_API_KEY`, `REPLICATE_API_TOKEN`, `UNSPLASH_*`, `PEXELS_*`, `REMOVE_BG_*`, R2 (5 vars), KV (5 vars auto-injetadas)
- Deploy `b0ff76f` Ready em 32s

### Bug crítico encontrado e corrigido
Durante o primeiro teste em produção, descobri que quando a API do Claude falhava, o **texto do erro era gravado como conteúdo do slide**. O JSON `{"type":"authentication_error","message":"invalid x-api-key"}` virou headline do carrossel do AgendaPRO.

**Causa raiz:** `/api/ai-copy-stream` emitia o erro no próprio body do stream (`\n[ERRO] msg`) e o cliente do editor escrevia letra-por-letra no slide via `updateSlide({ headline: clean })`. Como é stream, o HTTP status é sempre 200 — o check `!res.ok` não protegia.

**Fix:** sentinel único `\u0000<<IA_ERROR>>` no backend. Cliente detecta o sentinel, reverte o campo pro texto original e mostra alert. Zero contaminação de dados.

## O que aprendi

**Insight grande:** "Claude na plataforma" e "Claude API" são coisas separadas. Pagar o MAX dá acesso pro CLI no meu PC (tua conta logada, zero custo por mensagem) mas **não** dá acesso programático pra servidor. Pra app em produção rodar IA, tem que ter a API key separada com crédito separado.

**Custo real da API (medo era infundado):** Haiku 4.5 custa ~$0,07/R$0,35 por geração completa de carrossel. Pra meu volume realista (50-200 gerações/mês), fica entre **R$5-20/mês**. Muito longe do pesadelo dos R$550 do Replicate. O cost-tracker já tem hard cap de $10 programado.

**Aprendizado técnico:**
- Vercel Hobby tem cap de 60s por function. Rotas que tinham 90-120s foram reduzidas — se estourar, ou reduz prompt ou upgrade pro Pro.
- Upstash Redis plan Free (500k cmds/mês) é mais que suficiente pra app pessoal. SCAN cursor + MGET permite listagens eficientes.
- GitHub Contents API sem token tem rate limit de 60 req/h — com token (`GITHUB_TOKEN`), sobe pra 5000 req/h.
- Vercel lê env vars no runtime da função (cold start), não no build — adicionar env var e redeploy sempre necessário pra pegar.
- Streaming responses em Next precisam de sentinel pra distinguir erro de conteúdo — HTTP status não chega até o fim do stream.

- Sequência metodica: auditar → identificar blockers → migrar storage → renomear middleware → docs → deploy. Nada de pular etapa.

## Amanhã

### Resolver a key Anthropic (5 min)
1. https://console.anthropic.com/settings/keys → Revoke key atual → Create nova
2. Copiar no Notepad primeiro (começa com `sk-ant-api03-`, ~108 chars)
3. Vercel → Settings → Environment Variables → editar `ANTHROPIC_API_KEY` → paste
4. Deployments → último → Redeploy
5. Testar "gerar IA" num slide — deve funcionar

### Limpar carrossel corrompido (2 min)
O carrossel "como-o-agendapro-pode-ajudar-a-otimizar-seu-tempo" tem o JSON do erro 401 como headline/body. Opções:
- Apagar o texto nos campos e gerar IA de novo
- Deletar o carrossel inteiro e recriar
- Deixar como lembrete do bug corrigido

```
O que vamos criar hoje?
├─ Do zero (editor direto)

├─ De um template (galeria)
├─ Replicar post que vendeu (wins)
└─ Upload de referência (IA extrai estilo)
```

No editor: modo simples default (só slide + texto + gerar IA + navegar), avançado atrás de um toggle.
