# INBOX — Como usar

**Regra de ouro:** Captura SEM filtro. Não organize enquanto anota.

---

## Para que serve

Tudo que você aprende, observa ou quer lembrar vai aqui primeiro:
- Ideia que surgiu
- Padrão que percebeu num cliente
- Resultado de um teste
- Feedback recebido
- Decisão tomada

---

## Como anotar (template rápido)

```
Data: [hoje]
Projeto: [Impulso / MPN / UrbanFeet / Geral]
Captura: [o que aconteceu ou você percebeu]
```

---

## O que fazer depois

Quando a inbox tiver 5+ itens, processe:
- Se é padrão novo → move para `3-RETENCAO/padroes/`
- Se é estratégia → move para `3-RETENCAO/estrategias/`
- Se é tarefa de projeto → move para `2-PROCESSAMENTO/`
- Se não serve → deleta sem culpa
